## ✨ 리팩토링 개요
어떤 코드나 기능을 리팩토링할지 설명해주세요.

## 🔍 리팩토링 대상
- [ ] 코드 중복 제거
- [ ] 가독성 개선
- [ ] 성능 최적화
- [ ] 기타 (구체적으로 작성)

## 📌 현재 문제점
현재 코드의 어떤 부분이 개선이 필요한지 설명해주세요.

## 🎯 기대하는 개선 효과
리팩토링 후 기대하는 결과를 적어주세요.

## 📝 추가 내용
추가로 공유할 내용이 있다면 적어주세요.
